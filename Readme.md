# Heart Disease Prediction using PyTorch

## Overview
This project builds a binary classification model to predict the presence of heart disease using clinical features from the UCI Heart Disease dataset.

## Objective
To compare a PyTorch neural network against a logistic regression baseline for heart disease prediction.

## Folder Structure
- `data/` for raw and processed data
- `src/` for source code
- `notebooks/` for exploratory analysis
- `models/` for trained models
- `results/` for plots and metrics
